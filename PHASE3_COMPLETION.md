**목표**: 고양이 호감도 시스템과 카페 복원 시스템 구현

## ✅ 완성된 작업

### 1. CafeManager.cs (Cafe)
**책임**: 카페 복원 시스템의 핵심 관리

**구현 내용**:
- **21개 복원 작업** (3개 구역 × 7개 작업)
  - 구역 1: 입구 마당 (마당 빗자루질, 간판 고치기, 잡초 제거 등)
  - 구역 2: 카페 본채 (기와 올리기, 창호지 교체, 바닥 닦기 등)
  - 구역 3: 뒷마당 정원 (정원 정리, 연못 청소, 나무 가지치기 등)

**핵심 메서드**:
- `CompleteTask(taskId)` - 작업 완료 처리
  - 별 소비 확인
  - 보상 지급 (동전, 보석)
  - 구역 완료 확인
  - 고양이 언락 처리
- `GetTasksForArea(areaId)` - 구역별 작업 반환
- `GetAreaProgressPercentage(areaId)` - 구역별 진행도
- `GetCafeProgressPercentage()` - 전체 카페 진행도
- `IsAreaCompleted(areaId)` - 구역 완료 여부
- `GetNextIncompleteTask()` - 다음 미완료 작업

**이벤트**:
- `OnTaskCompleted` - 작업 완료 시
- `OnAreaCompleted` - 구역 완료 시
- `OnCafeUpdated` - 카페 업데이트 시

**특징**:
- 싱글톤 패턴
- 구역 완료 시 자동 고양이 언락
- 별 기반 작업 난이도 (2~14별)
- 점진적 보상 증가

---

### 2. RegularCustomerSystem.cs (Cafe)
**책임**: 단골 손님 시스템 관리

**구현 내용**:
- **5명의 단골 손님** (각 고양이에 대응)
  - 미라 (나비 선호)
  - 준호 (루나 선호)
  - 지은 (뭉치 선호)
  - 철수 (호두 선호)
  - 영희 (초코 선호)

**핵심 메서드**:
- `TryCustomerVisit()` - 손님 방문 시도
  - 일일 방문 제한 (최대 5명)
  - 선호 고양이 언락 확인
  - 자동 방문 처리
- `VisitCustomer(customer)` - 손님 방문 처리
  - 보상 지급 (동전, 보석)
  - 선호 고양이 호감도 증가
- `GetTodayVisitors()` - 오늘 방문한 손님
- `GetRemainingCustomersToday()` - 남은 방문 가능 손님
- `GetCustomersForCat(catId)` - 특정 고양이 선호 손님
- `ResetDaily()` - 일일 리셋

**이벤트**:
- `OnCustomerVisited` - 손님 방문 시

**특징**:
- 자동 방문 시스템 (5분 간격)
- 일일 최대 5명 방문
- 선호 고양이 기반 방문 조건
- 호감도 자동 증가

---

### 3. CafeUI.cs (UI)
**책임**: 카페 화면 UI 관리

**구현 내용**:
- **작업 목록 표시**
  - 작업 이름, 필요 별, 보상 표시
  - 완료/미완료 상태 표시
  - 완료 버튼 (상호작용)

- **고양이 표시**
  - 언락된 고양이 목록
  - 고양이 이름, 호감도 표시
  - 고양이 초상화 표시

- **진행도 표시**
  - 텍스트: "카페 복원 진행도: X/Y (Z%)"
  - 슬라이더: 시각적 진행도

**핵심 메서드**:
- `RefreshTaskList()` - 작업 목록 새로고침
- `RefreshCatDisplay()` - 고양이 표시 새로고침
- `UpdateProgressDisplay()` - 진행도 업데이트
- `OnCompleteTaskClicked(taskId)` - 작업 완료 클릭 처리

**이벤트 처리**:
- 작업 완료 시 UI 업데이트
- 카페 업데이트 시 모든 UI 새로고침

## 🎮 게임 플로우 통합

### 카페 시스템 플로우

```
[카페 화면 진입]
    ↓
[현재 진행도 표시]
    ├─ 전체 진행도 (%)
    ├─ 완료된 작업 수
    └─ 언락된 고양이 표시
    ↓
[작업 목록 표시]
    ├─ 각 작업의 필요 별, 보상
    └─ 완료 버튼
    ↓
[플레이어 작업 완료 클릭]
    ├─ 별 확인
    ├─ 보상 지급
    ├─ 구역 완료 확인
    └─ 고양이 언락 처리
    ↓
[단골 손님 방문]
    ├─ 5분 간격 자동 시도
    ├─ 선호 고양이 확인
    ├─ 보상 지급
    └─ 호감도 증가
    ↓
[카페 완료 시]
    ├─ 특별 보상
    └─ 엔딩 시퀀스
```
